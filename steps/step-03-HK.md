## （一）香港《虚拟资产服务供应商发牌制度》VASP（22项）

### 1. 持牌主体身份校验
- 检查网站首页是否披露持牌主体名称、SFC牌照编号、牌照类型、有效期
- **漏洞**: 持牌信息披露不完整 (CVSS 7.5)

### 2. 虚拟资产交易限额
- 检查是否设置交易限额、客户分层风控、高风险客户限制
- **漏洞**: 交易限额可绕过 (CVSS 9.0)

### 3. 大额交易自动上报
- 检查大额交易上报阈值、上报机制有效性
- **漏洞**: 上报机制无效或可绕过 (CVSS 9.0)

### 4. 可疑交易识别规则
- 检查可疑交易识别规则完善性、绕过可能性
- **漏洞**: 识别规则可绕过 (CVSS 9.0)

### 5. 客户资产独立托管
- 检查冷热钱包分账、客户资产与平台资产隔离
- **漏洞**: 客户资产未独立托管 (CVSS 9.5)

### 6. 客户资金与平台资金分离
- 检查资金分离逻辑、是否可挪用客户资金
- **漏洞**: 资金分离逻辑漏洞 (CVSS 9.5)

### 7. 资产储备证明防篡改
- 检查储备证明页面存在性、防篡改机制
- **漏洞**: 储备证明可篡改 (CVSS 9.0)

### 8. 平台保险披露
- 检查保险披露完整性、不可删除性
- **漏洞**: 保险披露不完整或可删除 (CVSS 7.0)

### 9. 杠杆/衍生品风险提示
- 检查强制风险提示弹窗、绕过可能性
- **漏洞**: 风险提示弹窗可绕过 (CVSS 7.5)

### 10. 客户交易授权二次确认
- 检查二次确认逻辑、防绕过机制
- **漏洞**: 二次确认逻辑可绕过 (CVSS 9.0)

### 11. 境外用户准入管控
- 检查地区IP拦截有效性、绕过可能性
- **漏洞**: 地区限制可绕过 (CVSS 8.0)

### 12. 交易历史留存
- 检查交易历史留存时间（≥7年）、安全性
- **漏洞**: 交易历史留存不足7年 (CVSS 7.5)

### 13. 监管数据调取接口
- 检查接口权限隔离、数据泄露风险
- **漏洞**: 接口权限隔离不足 (CVSS 9.0)

### 14. 业务变更事前通知
- 检查通知机制有效性、内容完整性
- **漏洞**: 通知机制无效 (CVSS 6.0)

### 15. 倒闭/破产客户资产取回
- 检查流程完善性、逻辑安全性
- **漏洞**: 取回流程不完善 (CVSS 8.0)

### 16. 广告营销风险警示

### 17. 佣金、返利活动合规

### 18. 第三方支付服务商接入

### 19. 链上交易溯源日志

## 检测方法

### 通用检测流程

```bash
# 1. 检查页面披露
curl -x 127.0.0.1:7890 -k -s https://$domain | grep -i "license\|牌照\|SFC"

# 2. 检查API端点
curl -x 127.0.0.1:7890 -k -s https://$domain/api/v1/compliance

# 3. 检查配置文件
curl -x 127.0.0.1:7890 -k -s https://$domain/config.json

# 4. 保存证据
curl -x 127.0.0.1:7890 -k -D headers.txt -o body.txt https://$domain/path
```

### 漏洞证据格式

```json
{
  "vuln_id": "VULN-VASP-001",
  "title": "持牌信息披露不完整",
  "severity": "high",
  "cvss_score": 7.5,
  "description": "网站未完整披露SFC持牌信息",
  "evidence": {
    "curl_command": "curl -x 127.0.0.1:7890 -k -s https://$domain",
    "response_headers": "...",
    "response_body": "..."
  },
  "recommendation": "在网站首页完整披露持牌信息"
}
```
